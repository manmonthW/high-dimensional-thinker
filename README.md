## 🎯 这是什么？

一个 Claude Code Skill，帮助你进行深度思考、本质分析和跨界创新。

**核心理念**：从"表面模式匹配"升级到"深层结构迁移"。

---

## 🔄 四步思维框架

| 步骤 | 名称 | 核心动作 |
|------|------|----------|
| 1 | **向量化拆解** | 将问题拆解为特征维度，找到隐含假设 |
| 2 | **升维寻根** | 找到支配现象的第一性原理 |
| 3 | **高维映射** | 跨界类比，用远距离领域重新审视 |
| 4 | **降维隐喻** | 用做菜/盖房/开车等生活场景解释 |

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/manmonthW/high-dimensional-thinker.git

# 创建符号链接到 Claude 技能目录
ln -s $(pwd)/high-dimensional-thinker ~/.claude/skills/high-dimensional-thinker
```

### 使用

在 Claude Code 中使用以下触发词：

- "帮我想清楚..."
- "分析一下本质..."
- "深度思考..."
- "拆解这个问题..."

或直接调用：`/high-dimensional-thinker`

## 💡 示例

### 输入
> 我是做产品经理的，最近看到AI产品经理、AI工程师这些新岗位，感觉自己的传统PM技能不够用了，但又不知道该学什么。

### 输出（摘要）

**向量化拆解**：
```
你的现状 = {
  当前身份: "传统产品经理",
  感知威胁: "AI PM、AI Engineer",
  隐含假设: "我需要成为AI专家才能生存"
}
```

**第一性原理**：产品经理的本质价值 = 翻译 + 决策

**跨界映射**：翻译官隐喻 — Google Translate 接管了词汇，但顶级翻译日薪过万，因为他们翻译的是"言外之意"

**降维隐喻**：导游 vs 地图软件 — 地图告诉你怎么走，导游告诉你值不值得去

**结论**：你的竞争力 = 你的判断力 - AI能替代的部分
